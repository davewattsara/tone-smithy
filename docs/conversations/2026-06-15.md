> when I up the resonance on filter 1 the sound gets too loud. it is at full in the red on the master tab

### [13:40:00] Claude (Opus 4.8)
Explained the cause: resonance maps to Q up to MAX_Q=25, and an LP SVF's resonant peak gain at the corner is ~Q (~+28 dB at full resonance), so harmonically rich input clips. Checked the DSP design doc, which explicitly wants self-oscillation ("the filter should sing"), so the fix is gain compensation, not a smaller resonance range. Added a `RESONANCE_GAIN_COMP` input attenuation that scales the filter input from x1.0 at resonance 0 down to x0.25 at resonance 1 — taming the peak while preserving self-oscillation and mimicking the classic ladder-filter body-thinning as resonance rises. Applied once at the input so both 12 and 24 dB/oct paths benefit. Added a test that the compensated peak stays well under the raw ~25x boost; all 178 engine tests pass, fmt + clippy clean. Offered to push the compensation stronger (0.75 -> ~0.85) if it still reads too hot.
